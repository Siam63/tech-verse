Place the contents of the folder "Team_02_01" into the htdocs folder of your XAMPP installation.

To import the database:
1. In phpmyadmin create a database called "scsstore" with collation "utf8mb4_unicode_ci".
2. Select the "scsstore" database and navigate to the Import tab.
3. Browse for the file scsstore.sql in the database folder and import the file.
